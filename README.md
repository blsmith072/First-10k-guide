This is a single-page landing page designed to sell a digital guide on saving your first $10,000. It features:

- **Premium Design** - Elegant color scheme with deep teal, gold accents, and cream backgrounds
- **Clear Value Proposition** - Immediately communicates what you'll get

## 🔧 Customization

To customize the page, edit these variables in the CSS:

```css
:root {
  --deep: #0F3D3E;        /* Main brand color */
  --gold: #C9A24B;        /* Accent color */
  --cream: #FBF8F1;       /* Background */
  --sage: #E4ECE6;        /* Section background */
}
```

To update content:
- Modify the Cash App link in both CTA buttons
- Update the chapters in the "What's Inside" section
- Edit the testimonial quote
- Change the price in `.price` spans
